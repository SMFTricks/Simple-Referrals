[size=4][color=#bf307a]Introduction[/color][/size]
[size=1em]Simple Referrals implements a referrals system to your forum.[/size]

[size=3][color=#1f4882]Features[/color][/size]
[list]
[li][size=0.9em]Set a referral using autosuggest in the registration form[/size][/li]
[li][size=0.9em]Use a referral url with the referral id for easy share[/size][/li]
[li][size=0.9em]Display referral count in profile[/size][/li]
[li][size=0.9em]Display referral count in posts[/size][/li]
[li][size=0.9em]Display referral link in their profile[/size][/li]
[li][size=0.9em]Enable/disable the selection of searching users in the registration form[/size][/li]
[li][size=0.9em]Maintenance task for recounting the referral count of each user in the forum[/size][/li]
[li][size=0.9em]Top referrers statistics[/size][/li]
[/list]
